# Python-Password-Generator-
A simple Python script that generates secure, random passwords. It asks the user for a length (minimum 12 characters) and then mixes letters, numbers, and symbols to create a strong, unpredictable password.

Input Validation: The script uses a while True loop to keep asking for a password length until you enter a number that is 12 or higher.

Character Pool: It combines uppercase letters, lowercase letters, digits, and symbols to create a strong password.

Random Generation: Using a for loop, the script picks random characters from the pool until it reaches your desired length.
